# Banco de Dados

## Quais os bancos de dados suportados?

- DB2;
- Firebird;
- Oracle;
- Microsoft SQL Server;
- PostgreSQL;
- MySQL .

## Como funcionam as tarefas de gerenciamento do banco pelo Maker?

No Maker, existe o Executor de Script SQL, onde é possível executar as operações básicas, bem como criar functions, procedures, etc.

E para as seleções, junções, uniões, etc, existe o Assistente SQL, que é um utilitário visual para auxílio na construção de consultas em linguagem SQL.

## Existe alguma interação com ferramentas Case para modelagem de dados?

Essa interação não existe, o Maker não dispõe de qualquer interação com outra ferramenta de auxilio a banco de dados.

## É necessário instalar e configurar aplicações baseadas no modelo Cliente/Servidor?

A instalação do banco de dados é feita no servidor. Ele é que faz o envio dos dados para os clientes conectados. Esse modelo é utilizado juntamente com a tecnologia Java, integrado com os protocolos HTTP e SOAP.
