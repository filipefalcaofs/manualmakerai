# Criando atalhos do sistema na área de trabalho

Este recurso permite que os usuários criem atalhos das telas e relatórios que acessam com mais frequência. Os atalhos são armazenados de acordo com a identificação dada pelo usuário.

Para criar atalhos, siga os passos:

1. [Acesse o Webrun](/docs/webrun/acessando_webrun).
2. Navegando pelo menu do sistema ativado, posicione o mouse sobre a opção para a qual deseja criar o atalho, arraste e solte-o na área de trabalho do Webrun.
3. Para acessar o objeto, execute clique duplo sobre o atalho.

Para excluir um atalho, selecione o objeto e utilize a tecla DELETE. Confirme a exclusão.
