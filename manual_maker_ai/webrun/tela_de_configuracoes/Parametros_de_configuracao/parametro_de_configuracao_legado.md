# Parâmetros de Configuração - Legado

O grupo Legado permite configurar parâmetros que mantêm retrocompatibilidade com algum recurso anterior.

![Legado](/assets/images/legado_01-98281d4e305399258bb91024c34f4613.png)

Abaixo, verifique o principal parâmetro para o grupo **Legado**:

- ApiDeMensagem - Caso definido como ***true***, o Webrun exibirá as mensagens da API de Mensagens (Objeto Interação) no modelo anterior (Versão 1.2.0.45-). Por padrão, a partir da versão 1.2.0.46 do Webrun, as mensagens são exibidas no padrão SweetAlert2.
