# Pasta CONFIG

> Fonte: https://manual.softwell.com.br/docs/webrun/estrutura_de_pastas_e_arquivos/pasta_config

[## 📄️ Arquivo ADMIN.XML

Neste arquivo, é possível fazer a alteração do Login e da Senha do administrador que tem acesso às Configurações no Webrun. Este arquivo encontra-se na pasta C:\Arquivos de programas\Maker Studio Bootstrap\Webrun Studio Bootstrap\config.](/docs/webrun/estrutura_de_pastas_e_arquivos/Pasta_CONFIG/arquivo_admin_xml)

[## 📄️ Arquivo CONFIG.XML

Este arquivo registra as configurações definidas na tela Parâmetros de Configuração.](/docs/webrun/estrutura_de_pastas_e_arquivos/Pasta_CONFIG/arquivo_config_xml)

[## 📄️ Arquivo DATABASES.XML

As declarações das chaves de conexão encontram-se nas tags group. Este arquivo está composto pelas chaves de conexão com os bancos de dados que o Webrun suporta.](/docs/webrun/estrutura_de_pastas_e_arquivos/Pasta_CONFIG/arquivo_databases_xml)
