# Pasta SYSTEMS do Webrun

Pasta onde encontramos os arquivos \*.WFRE, arquivos de projetos criados no Maker.

Deve-se observar que, caso um dos arquivos de projeto esteja corrompido, o Webrun não abre. Para localizar o defeito, digite o endereço http://(servidor):(porta)/webrun/autotest e, na tela, será apresentado o erro (arquivo danificado que deverá ser substituído pelo mesmo arquivo com extensão \*-dist.xml - localizado na pasta C:\Arquivos de programas\Softwell Solutions\Maker Studio Bootstrap\Webrun Studio Bootstrap\config).

Nesta pasta, podemos personalizar o Webrun, com a criação de uma estutura de pastas iniciando por Maker.Commons.
