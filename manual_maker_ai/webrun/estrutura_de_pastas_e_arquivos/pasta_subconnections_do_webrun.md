# Pasta \_subConnections do Webrun

A pasta Systems (C:\Arquivos de programas\Softwell Solutions\Maker Studio Bootstrap\Webrun Studio Bootstrap\systems) armazena todos os wfre dos sistemas desenvolvidos no Maker. Se em ao menos um sistema houver conexões adicionais, será criada uma pasta chamada \_subConnections que servirá para armazenar outras subpastas, nomeadas com a sigla de cada sistema, contendo arquivos wfre que apontam para as conexões adcionais configuradas.
