[## 📄️ MasterSkin

O MasterSkin é a solução de personalização visual para sistemas desenvolvidos dentro da plataforma Maker, a partir da personalização das propriedades nativas do framework Bootstrap. Focado em agilidade, padronização e controle, ele permite configurar toda a identidade visual de uma aplicação de forma centralizada e segura, sem a necessidade de editar diretamente o código fonte.](/docs/webrun/tela_de_configuracoes/master_skin)

[## 📄️ Progressive Web App - PWA

O Progressive Web App (PWA) é uma maneira de trazer a sensação de aplicativo nativo para um aplicativo da Web tradicional, dando capacidade de instalá-lo na tela inicial de um dispositivo móvel ou desktop, para acessar a câmera do dispositivo, receber notificações push e fazer a sincronização em segundo plano.](/docs/webrun/tela_de_configuracoes/pwa)

[## 📄️ Tela de Configurações do Webrun

Na Tela de Configurações do Webrun é possível definir parâmetros de configuração, adicionar e remover sistemas da lista, recarregar sistemas, visualizar a lista de Web Services e estilizar a interface visual do Webrun.](/docs/webrun/tela_de_configuracoes/tela_configuracoes)
