[## 📄️ Estrutura de pastas do Webrun

A estrutura de pastas de configuração do Webrun é composta pelas pastas CONFIG, JAVA, REPORTS, SYSTEMS, TEMP, TOMCAT e UPDATES, localizadas em C:\Arquivos de Programas\Softwell Solutions\Maker Studio Bootstrap\Webrun Studio Bootstrap .](/docs/webrun/estrutura_de_pastas_e_arquivos/estrutura_de_pastas_do_webrun)

[## 📄️ Pasta JAVA do Webrun

Pasta da máquina virtual JAVA. Versão 8. Caso o usuário queira utilizar outra versão, deve substituir o conteúdo desta.](/docs/webrun/estrutura_de_pastas_e_arquivos/pasta_java_do_webrun)

[## 📄️ Pasta REPORTS do Webrun

A pasta REPORTS é o diretório que contém os arquivos necessários para a geração do relatório, tendo como principal arquivo o WFRReports.exe.](/docs/webrun/estrutura_de_pastas_e_arquivos/pasta_reports_do_webrun)

[## 📄️ Pasta \\_subConnections do Webrun

A pasta Systems (C:\Arquivos de programas\Softwell Solutions\Maker Studio Bootstrap\Webrun Studio Bootstrap\systems) armazena todos os wfre dos sistemas desenvolvidos no Maker. Se em ao menos um sistema houver conexões adicionais, será criada uma pasta chamada \_subConnections que servirá para armazenar outras subpastas, nomeadas com a sigla de cada sistema, contendo arquivos wfre que apontam para as conexões adcionais configuradas.](/docs/webrun/estrutura_de_pastas_e_arquivos/pasta_subconnections_do_webrun)

[## 📄️ Pasta SYSTEMS do Webrun

Pasta onde encontramos os arquivos \\*.WFRE, arquivos de projetos criados no Maker.](/docs/webrun/estrutura_de_pastas_e_arquivos/pasta_systems_do_webrun)

[## 📄️ Pasta UPDATES

Localizada em C:\\Arquivos de programas\\Softwell Solutions\\Maker Studio Bootstrap\\Webrun Studio Bootstrap\\updates, é o diretório onde encontramos a atualização automática do Webrun. Ele está composto pelos arquivos update.exe, update.URS, update.Lang, e pelo arquivo de atualização webrun.war.](/docs/webrun/estrutura_de_pastas_e_arquivos/pasta_updates)
