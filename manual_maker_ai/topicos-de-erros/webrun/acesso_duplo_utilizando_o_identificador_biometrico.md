# Acesso duplo utilizando o Identificador Biométrico

![image1771.jpg](/assets/images/dis1-8e448c5221681afa5a0bdbcf7bb79b08.png)

Causa: Ao tentar utilizar o [Identificador Biométrico](/docs/maker/instalacoes/instalacao_servidor_biometrico) no mesmo browser para mais de um sistema que estejam abertos.

Solução: Para que funcione corretamente, é necessário reiniciar o browser antes de acessar novamente o applet do outro sistema.
