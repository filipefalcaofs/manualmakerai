# IO exception usando o banco de dados SQL SERVER 2008.

Erro: Erro no webRun, IO exception usando o banco de dados SQL SERVER 2008

Solução: É preciso configurar o banco de dados para que o webrun possa acessá-lo, verifique os passos clicando [aqui](/docs/dicas-e-truques/banco_de_dados/como_configurar_conexao_remota_sqlserver2005).
