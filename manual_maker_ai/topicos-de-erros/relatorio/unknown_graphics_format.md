# Unknown Graphics format

Causa: Gráfico do relatório sem imagens compatíveis com o gerador de relatórios.

Solução: Ao gerar relatórios no Webrun com imagens do tipo .PNG e .GIF salvas no banco de dados, apresenta-se a mensagem de erro: Unknown Graphic Format. Para gerar o relatório com imagens utilize com extensões BMP ou JPEG.
