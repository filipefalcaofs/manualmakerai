# Criação de Projetos Simplificada

# Facilidade na criação de projetos

Pule etapas manuais e comece a desenvolver com o banco de dados configurado automaticamente

## Visão Geral

A **Criação de Projetos Simplificada** permite criar um novo projeto sem precisar configurar manualmente os parâmetros de conexão com o banco de dados.

Ao usar essa opção, o Maker define automaticamente a conexão com o banco **Postgres** que já vem instalado e configurado na instalação padrão do sistema.

Se o usuário preferir, ainda é possível clicar em **Avançado** e ajustar as configurações de conexão manualmente.

## Como Funciona

1. Na tela inicial do Maker, clique em **Novo Projeto**.
2. Insira o nome, a descrição e o template do projeto.
3. Confirme e pronto! O projeto será criado com a conexão ao banco de dados já configurada.

## Principais Benefícios

### ⚡ Rapidez na Iniciação

**Menos Passos**: Não é necessário preencher dados de conexão manualmente.

**Pronto para Desenvolvimento**: O projeto já vem configurado para o banco padrão.

### 🛠️ Flexibilidade

**Configuração Avançada**: Possibilidade de personalizar parâmetros de conexão.

**Compatibilidade**: Funciona tanto para quem quer rapidez quanto para quem precisa de ajustes específicos.

### 📉 Redução de Erros

**Menos Configurações Manuais**: Evita erros comuns na definição de parâmetros.

**Conexão Garantida**: Usa o banco de dados já testado e homologado na instalação.

### 📈 Produtividade

**Agilidade**: Início imediato do desenvolvimento.

**Fluxo Otimizado**: Menos tempo gasto com configuração inicial.

### Por Que Usar?

1. **Ganhe tempo** — crie projetos prontos em segundos
2. **Evite erros** — conexão configurada automaticamente
3. **Tenha flexibilidade** — personalize quando necessário
