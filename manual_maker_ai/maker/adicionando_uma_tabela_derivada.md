# Adicionando uma tabela derivada

Semelhante ao item Adicionando uma subconsulta, também é possível criar tabelas derivadas no Assistente SQL com um clique com o botão direito do mouse sobre a Área de Construção de Consultas, selecionando a opção Adicionar tabela derivada. O uso desse recurso tem como resultado um tipo de elemento, um tipo linha, e seus valores se baseiam nos valores das tabelas subjacentes quando a expressão de consulta é avaliada.

1. Clique com o botão direito do mouse sobre a área da aba Principal e selecione do menu pop-up a opção Adicionar tabela derivada:

![image1210.jpg](/assets/images/add_tabela_derivada1-8b6b232dff2300ab1633f6faaff8a89b.png)

2. O usuário pode inserir várias tabelas derivadas. Para cada inserção, será atribuída uma nova aba para a constituição da consulta da tabela derivada:

![image1211.jpg](/assets/images/add_tabela_derivada2-7c28a49b960ef13e4087038236a4f19f.png)

3. Na nova aba (Query1, por exemplo), deve-se constituir a consulta:

![image1212.jpg](/assets/images/add_tabela_derivada3-9d550ff3d10d35aabb19573f7d6622d3.png)

4. Retornando à aba Principal, selecione os campos da tabela derivada:

![image1213.jpg](/assets/images/add_tabela_derivada4-206c73544bc15acf7d148e2c199beda0.png)

5. Visualização da consulta pela aba SQL:

![image86.jpg](/assets/images/add_tabela_derivada5-a91121d077e597f51a2ce8d94c75b95c.png)
