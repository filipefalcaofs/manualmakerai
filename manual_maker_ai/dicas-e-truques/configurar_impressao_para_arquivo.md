# Configurar Impressão para Arquivo

Este recurso gera um arquivo-texto a partir de campos selecionados de um relatório.

Siga os passos:

1. Acesse o Maker Reports e abra um relatório;
2. Ative o menu Arquivo e, deste, a opção Configurar Impressão para Arquivo;

   Na tela, defina:

   1. No campo Nome do Arquivo - informe o nome do arquivo que será gerado e o local;
   2. Para o campo Tipo do Arquivo, defina o separador dos campos;
   3. Delimitado por vírgula;
   4. Delimitado por Tabulação;
   5. Delimitado por Tamanho Fixo.
   6. No campo Faixa, selecione a(s) banda(s) de dados e, logo abaixo, no campo Controles disponíveis, selecione os campos de dados.

Utilize os controles de ordenação para orientar a sequência dos campos.

A geração do arquivo ocorre na execução da visualização do relatório pelo Webrun e seu conteúdo pode ser visualizado por um editor de texto (por exemplo, Bloco de Notas do Windows).
