# Tela de Notificação

O Maker Mobile disponibiliza uma tela de notificações, onde cada problema na exportação do projeto será notifcado nesta tela.

- Ao ser exportado o projeto para a plataforma desejada, será aberta a seguinte tela:

![](/assets/images/notificacao_1-44a4aa2ca9f9f248ad01a5ce8ba9b468.jpg)

- Observe que ao selecionar um projeto o botão de Notificação será habilitado.

![](/assets/images/tela_notificacao_2_-5ad0bc49df2b5300c868d0f1a4ffadaa.jpg)

- Clique sobre o botão e observe as notificações:

![](/assets/images/tela_notificacao_3-9a3af47d9eca38209e6e513f94edff1a.jpg)

Observe que as algumas funções não foram encontradas, isso ocorre por que no projeto houve funções que não são compatíveis com o Maker Mobile. Por isso é muito importante verificar no desenvolvimento da aplicação se a função utilizada é compátivel, pois exportando uma função "errada" a aplicação pode não ser executada corretamente.
