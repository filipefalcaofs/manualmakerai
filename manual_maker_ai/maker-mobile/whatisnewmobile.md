# O que há de novo? - Maker Mobile

#### Novidades Maker Mobile

**Consulte o histórico de alterações  [clicando aqui.](/docs/maker-mobile/alteracoesmobile)**

API de Funções: 25/06/2019

- **Waze - Abrir**

Ótimo para levar o usuário direto da sua aplicação para o Waze, tornando uma forma ágil de trocar de aplicação.

- **Waze - Inicializar SDK e navegar**

Com essa função é possível utilizar a navegação do Waze dentro do próprio aplicativo, assim como é utilizado por aplicativos de transporte (99, Uber, ItMov...).

- **Waze - Fechar Conexão**

Permite finalizar a conexão com o Waze após a sua utilização.

- **Waze - Obter Versão**

Obtém a versão do Waze utilizada, possibilitando verificar a compatibiliza da aplicação.

- **Firebase - GeoFire - Definir Local**

Define a localização do dispositivo utilizado para poder ser usado em funções de Georreferenciamento

- **Firebase - GeoFire - Monitorar Área**

Permite verificar o fluxo de usuário de determinada área, como ocorre por exemplo, quando os usuários solicitam os motoristas por aplicativos de transporte e conseguem visualizar os carros próximos.

- **Firebase - GeoFire - Parar Monitoramento da Área**

Remove o monitoramento da área

- **Compartilhar Texto**

Possibilita ao usuário mandar links ou textos de um aplicativo para outro.

- **Enviar Arquivos via POST Assíncrono**

Com essa funcionalidade será possível enviar arquivos via POST sem que a aplicação pareça travada para o usuário, e permitindo que o mesmo utilize a aplicação enquanto o envio é feito.

- **Escolha Arquivo**

Com essa função o usuário poderá navegar pelo diretório de arquivos do aparelho e selecionar um arquivo. Será passado o caminho desse arquivo para um fluxo possibilitando a manipulação seja para envio ou apenas leitura.

- **Mobile - Abrir URL no Contexto**

Abre uma url em outra janela, evitando a nova janela se sobreponha a quem a invocou e que as ações feitas sejam perdidas.

- **Obter Informações do Dispositivo**

Uma de suas utilidades é poder oferecer assistência a um cliente, ou simplesmente entender quais tipos de dispositivos um público alvo utiliza entre outras utilidades obtendo informações como: Versão do Sistema Operacional, Modelo do dispositivo, Tipo do Sistema operacional (Android ou iOS).

- **Obter Versão do Aplicativo**

Informa em qual versão está o aplicativo, útil para saber se o mesmo está atualizado.

- **Firebase - Autenticação Email**

Realiza a autenticação do usuário através de e-mail e senha, caso não esteja cadastrado uma nova conta será criada para este usuário. Funcionalidade que garante agilidade para os usuários, evitando a demora que existe em muitos cadastros.

- **Firebase - Autenticação Personalizada**

Realiza a autenticação do usuário utilizando apenas um token personalizado gerado pelo retaguarda, garantindo assim maior segurança.

- **Firebase - Usuário Logado?**

Verifica se há alguma sessão ativa do Firebase para o aplicativo que chamar a função.

- **Firebase - Deslogar**

Esta função efetua logout do usuário autenticado no Firebase. Caso esta função seja chamada através de aplicativos móveis (Maker Mobile), a aplicação será encerrada.

- **Firebase - Redefinir senha**

Permite redefinir a senha do e-mail cadastrado na lista de usuários do projeto do Firebase.

- **GPS - Iniciar monitoramento**

Essa função inicia o monitoramento de localização, passando para o fluxo escolhido seus dados.

- **GPS - Desativar serviço de localização**

Essa função encerra a conexão com o serviço GPS.

- **GPS - Iniciar serviço de localização**

Essa função inicia o serviço de localização do dispositivo.

- **GPS - Obter coordenada atual**

Esta função retorna a localização atual do dispositivo.

- **Mapa - Obter Objeto a partir das coordenadas**

Retorna objeto da rota traçada com seus atributos

- **Mapa - Obter Objeto a partir das coordenadas**

Retorna objeto da rota traçada com seus atributos
