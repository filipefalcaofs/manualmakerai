# Formulários Maker Mobile

Atualmente, os componentes listados abaixo não são suportados pela plataforma Mobile.

- **Gráfico**
- **Lista Dupla**
- **Subform**
- **Árvore**

Observações:

- O tamanho e o posicionamento dos componentes no Maker Mobile utilizam valor percentual, dessa forma não é possível determinar um tamanho fixo, pois o mesmo se ajusta de acordo com o tamanho da tela. Para configurar o redimensionamento selecione as opções Largura e altura ou Apenas Largura na Lookup da tela de exportação.
- O componente **Lista Dinâmica** apesar de ser funcional possui algumas restrições no **Maker Mobile**, observe abaixo algumas delas:
  - O componente não possui pesquisa;
  - Não aceita chave composta;
- A propriedade paginação não é funcional.
- Componente **Botão** mesmo sem evento associado ao evento Ao Clicar, terá comportamento semelhante a de um  
  Botão com evento.
