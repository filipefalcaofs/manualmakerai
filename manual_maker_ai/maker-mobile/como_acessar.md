# Como acessar o Maker Mobile

A utilização do Maker Mobile é muito simples.

- Para utilizá-lo basta acessar o menu do Maker -> *Mobile* -> *Login*.

  ![](/assets/images/exportacao_1_-715c32a8926c69d3da414ed0586fa831.jpg)
- Com a tela de autenticação aberta, basta preencher os campos **Usuário** e **Senha** e clicar em **Conectar**.

  ![](/assets/images/exportacao_2_-fea72258b70daa6419ae2f1c371fc1d5.jpg)

#### Criando uma nova conta

- Para o primeiro acesso, é necessário que o usuário seja cadastrado. Utilize a opção *Cadastre-se* da tela de Login para acessar esta tela.

  ![](/assets/images/exportacao_3_-1b567abc42341eec8c5c6f581dba14ea.jpg)
- Preencha todos os dados e clique em **Próximo**. Na próxima tela, preencha os dados solicitados.

  ![](/assets/images/exportacao_4_-8c677cbc97f6652667334bba995e09c5.jpg)

  > Caso as informações estejam corretas e a chave de licença tenha acesso ao Maker Mobile, o login será feito automaticamente no ambiente mobile.
- Caso a autenticação seja realizada com sucesso, os menus [Exportar para plataforma](/docs/maker-mobile/exportando_projetos/como_exportar), [Lista de projetos em compilação](/docs/maker-mobile/exportando_projetos/lista_exportacao), [Comprar Créditos](/docs/maker-mobile/comprando_creditos_para_exportacao_de_projetos) e **Histórico de compras** serão habilitados e os recursos e propriedades suportadas pelo Mobile estarão destacados.

  ![](/assets/images/exportacao_5_-c9fcfa3c233b7692cf44736b58f4287c.jpg) ![](/assets/images/exportacao_6_-e85bd6a12742234d62189d9e5b15665c.jpg)

  Observe que um ícone de um aparelho mobile aparecerá nas propriedades de componentes e em funções que forem compatíveis com o Maker Mobile.
