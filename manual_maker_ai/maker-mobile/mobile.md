# Maker Mobile

Em todo o mundo a mobilidade corporativa, através do uso primeiro dos palms, depois dos smartphones e agora dos tablets, deixou de ser apenas uma forte tendência tecnológica para tornar-se um novo vetor de crescimento e expansão empresarial. Hoje, grande parte das empresas tem praticado algum nível de mobilidade através do uso desses equipamentos em seus processos de trabalho, negócios e na relação com os seus clientes. Para atender a esses novos paradigmas do mundo corporativo, a Softwell Solutions criou o Maker Mobile, uma ferramenta para construção de aplicações de software para dispositivos móveis, que utiliza a plataforma Maker como ambiente de desenvolvimento. Em função do seu alto nível de abstração tecnológica o Maker Mobile permite um alto ganho de produtividade ao programador, na medida em que simplifica os processos de desenvolvimento hoje utilizados pelas empresas, gerando consequentemente grande competitividade para o segmento corporativo.

**Funcionalidades do produto:**

- IDE Maker adaptada para as características das aplicações móveis (Layout de formulários, componentes visuais, acesso a dados, dentre outras peculiaridades);
- Regras de negócio representadas em fluxograma;
- API de funções especializada para utilização dos recursos de dispositivos como GPS, Câmera Digital, Wi-Fi, Bluetooth, Microfone, Biometria;
- Suporte a banco de dados embarcados (SQLite), MongoDB, Firebase;
- Sincronização de dados.

**Plataformas suportadas:**

- **iOS** - Versão mínima 9.0;
- **Android** - Versão mínima 4.4 (KitKat);

# Exemplos

**Aplicativos de exemplo desenvolvidos no Maker Mobile que demonstra o uso das diversas funcionalidades existentes no Maker.**

![](/assets/images/exemploFreelancerLogo-08f71882953947dba4bc6def0bc0e12c.png)  
**Encontrar Free Lancer**

Aplicativo de exemplo destinado a localização de Free Lancer. Neste aplicativo usamos poucas das diversas funcionalidades que o Maker Mobile proporciona, entre elas usamos: Componente Integração, API de Mapa, Localização, Utilização de Marcadores, Persistência de Dados, Integração com o Framework7.

Com este aplicativo você poderá localizar rapidamente um profissional Free Lancer. Encontre por exemplo: diaristas, eletricistas, pedreiros, dentre diversos profissionais cadastrados.

**Minhas Tarefas**

Aplicativo de exemplo destinado para o controle de atividades pessoais do dia-a-dia. Utiliza as seguintes funcionalidades: Componente Integração, Persistência de Dados, Integração com o Framework7.

Com este aplicativo você poderá controlar e monitorar as atividades pessoais, podendo iniciar, finalizar e visualizar as atividade.

**Coopera Jacobina**

Aplicativo de exemplo destinado à integração da população com a administração do município. Utiliza as seguintes funcionalidades: Componente Integração, Persistência de Dados, Integração com o Framework7, API de Mapa, Localização, Utilização de Marcadores, API do Facebook, Personalização de Componentes através de Framework's.

O Coopera Jacobina é uma iniciativa da Prefeitura que precisa da sua participação ativa, cidadão. Colabore mantendo a cidade limpa e preservando os equipamentos e espaços públicos, fiscalize as ações e obras da Prefeitura, baixe e use o 'Coopera Jacobina' no seu celular e envie suas queixas ou solicitações.
