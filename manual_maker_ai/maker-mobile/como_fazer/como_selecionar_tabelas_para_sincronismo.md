# Selecionando tabelas para sincronismo

Após ter efetuado por completo o [cadastro do sistema no Gerenciador](/docs/maker-mobile/como_sincronizar), acesse o menu **Sincronismo** em seguida **Criar tabelas de sincronismo**.

![](/assets/images/como_selecionar_tabelas_para_sicronismo-b8da2d0fae8117a6529d4a1edfa999d3.jpg)

Com o formulário de sicronismo aberto, selecione um sistema e em seguida o botão Carregar arquivos de Sincronização onde será apresentada uma tela para upload do arquivo selecionado na exportação do projeto.

![](/assets/images/como_selecionar_tabelas_para_sicronismo_2-67a791113f0a648c0b307e8c299df513.jpg) ![](/assets/images/como_sicronizar_tabelas_6-33333cb9da83c05ea5a02ec8290a433c.jpg)

Após a seleção dos dados, clique sobre o botão e o sincronismo estará pronto.

![](/assets/images/como_sicronizar_6-bb926864157edc597d72dc7e7e3e9933.jpg)
