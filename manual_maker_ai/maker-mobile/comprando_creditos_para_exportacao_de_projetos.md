# Comprando créditos para exportação de projetos

![](/assets/images/comprando_creditos-739f5aeeb9a6f066b1d5306e5260a5f3.jpg)

Como comprar créditos?

Para comprar créditos para o Maker mobile, siga os passos que serão apresentados abaixo.

> **Observação**: O usuário deverá estar cadastrado no Maker Mobile. Qualquer dúvida sobre cadastro no Maker mobile, clique [aqui](/docs/maker-mobile/como_acessar#novaconta).

- Primeiro, vá até o menu Maker -> *Mobile* -> *Comprar Créditos*.

![](/assets/images/comprando_creditos2-c2cef4d1ee9a5a4e3916ee24a189009e.jpg)

- Ao selecionar a compra de créditos será aberta uma tela, onde deverá ser selecionada a plataforma, o pacote da compra e o tipo de pagamento. Neste exemplo será utilizado o método do PagSeguro. Então, selecione PagSeguro e em seguida Realizar Pagamento.

![](/assets/images/comprando_creditos3-1acd2a92397df1c1b8eb136dec356ac4.jpg)

- Feito os passos anteriores, o usuário irá para a página do Pagseguro, onde deverá informar seus dados:

![](/assets/images/comprando_creditos5-697407c70c9975827e17fcab77d12665.jpg)

- Caso não possua cadastro no PagSeguro, informe o seu e-mail e clique no botão "Pagar com segurança":

![](/assets/images/comprando_creditos6-b36b024d2046d6b2db403f7141f8a4ec.jpg)

- Insira seus dados de endereço:

![](/assets/images/comprando_creditos7-501eb3222d3de5f2306f8d46c40683fb.jpg)

- Abaixo, você poderá escolher uma das formas de pagamento que o PagSeguro disponibiliza:

![](/assets/images/comprando_creditos8-8ae8ee667f5aab7ca78bd5545a4fca3e.jpg)

- Selecione o pagamento desejado, efetue o pagamento e desfrute do pacote desejado.
