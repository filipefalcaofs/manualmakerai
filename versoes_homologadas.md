# Versões Homologadas

### Banco de dados

- [PostgreSQL 9.5/10/11/12/13/14](/docs/banco-de-dados/postgresql)
- [MySQL 5.6/5.7/8.0 e MariaDB +10](/docs/banco-de-dados/mysql_server)
- [DB2 9.5](/docs/banco-de-dados/utilizando_db2_no_maker)
- [Firebird 2.5.9/3.0/4.0](/docs/banco-de-dados/utilizando_o_firebird)
- [Oracle 10g/11g/12c/19c/21c](/docs/banco-de-dados/oracle)
- [MS SQL Server 2008R2/2012/2014/2019](/docs/banco-de-dados/ms_sql_server)

### Navegador

- Google Chrome (em suas versões mais atuais).
- Mozilla Firefox (em suas versões mais atuais).
- Microsoft Edge.

### Sistema Operacional

- Windows 7, 8.0, 8.1, 10, 11 (Para Maker e Webrun).
- Windows Server 2008, 2003, 2012 (Somente Webrun).
- Linux (Somente Webrun). Exemplos: Ubuntu, Red Hat, Slackware, Debian, SuSE, Fedora, Mandriva, entre outros....

### Servidor de Aplicação

- JBoss-4.2.1GA.
- Webshere Application Server (6.1 e 7 e 8.5 em Fase de Homologação) .
- Tomcat6, Tomcat7 e a partir da versão Maker Studio, Tomcat 8.5 e Tomcat 9.
- IIS 6.0 , 7.5 e 8.5.
- Dot.Net Framework 4.0 (A partir do Maker 3.6) e 2.0 (Versões inferiores do Maker 3.5).

### JDK

- JDK6 (1.6)
- JDK7 (1.7).
- JDK8 (1.8) para versões Maker Studio ou superiores.
